CRAP Principles

The CRAP principles—Contrast, Repetition, Alignment, and Proximity—are core design concepts used to improve visual communication in web design.

Contrast helps important elements stand out, while repetition creates consistency by reusing visual styles throughout a site. Alignment ensures that elements are visually connected and organized, giving a cleaner appearance. Proximity groups related elements together, helping users understand the relationship between different parts of the content.

Color Guides

Color guides help designers choose effective color palettes that enhance usability and aesthetics.

Using color theory, complementary and analogous schemes can be built for harmony and contrast. Tools like the color wheel, accessibility checkers, and contrast ratios are essential when selecting colors that work across devices and for users with visual impairments. CSS supports defining colors using keywords, HEX codes, RGB, HSL, and more, giving designers flexible ways to style elements.

CSS Positioning

CSS positioning allows developers to control exactly where elements appear on a webpage.

The five primary positioning values are static, relative, absolute, fixed, and sticky. Each value affects how an element is placed in the flow of the document or relative to its parent or the viewport. For example, absolute positioning removes the element from the normal flow and positions it based on its nearest positioned ancestor, while fixed keeps it locked to the viewport. Mastering positioning is key for creating dynamic and responsive layouts.

Box Model Introduction

The CSS box model describes how elements are rendered on a webpage.

Every element is essentially a rectangular box composed of four parts: content, padding, border, and margin. Understanding how these parts interact is essential for layout and spacing. For example, padding creates space inside an element, while margin adds space outside. Misunderstanding the box model can lead to layout bugs, especially when using widths and heights with borders and padding.
