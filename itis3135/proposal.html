Design Proposal Webpage

Project Overview:

I plan on making a website to educate users on the basics of CSS. This page would be intended for beginner computer science sutdents and it will work as an introduction to CSS intended to be viewed before learning CSS.

Client Information:

My client is a peer from Univsersity who would like to learn more about web development.

Site Map:

Page Design:

Home:

This will be the home page of the website and it will serve as a small introduction and include naviagtion links to the rest of the website. The audience will be the viewer and it will not include much functionality other than the naviation links.

About Us:

This page will serve as an introduction to myself and my client, the purpose of this page and how to contact us. The audience will be the viewer and it will include functionality to contact us.

Desgin Guides:

This page will have an overview of common CSS design principles and will focus more on the actual appearance of websites rather than the code used to achieve that appearance. It will not include much functionality other than navigation links. This page and all others after it are intended for the viewers of the page.

Crap Principles:

This page will cover just the crap principles outlined to us in this course.

Box Model:

This page will be an overview of the CSS box model, why it is used and how it works. It will include some functionality to move and show the boxes on the page as well as to navigate to other pages.

Syntax:

This page will serve as an overview of CSS syntax and will feature the most important rules and things that often cause confusion when writing CSS. It will only have functionality to navigate to other pages.

Examples:

This page will show creative examples of what can be done with CSS, some of which will be interactive, and will include more navigation links.

Helpful Links:

This page will include previews of other websites which are helpful for beginner CSS users as well as nagivation links.

Dynamic Functions:

Some of the dynamic functions which will be featured on this page include navigation links, self editing pages and customizable pages.
